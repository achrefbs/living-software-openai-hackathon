# Submission checklist

Deadline: **Tuesday, July 21, 2026 at 5:00 PM PT** / **Wednesday, July 22 at 02:00 Europe/Madrid**.

## Repository

- [x] Separate challenge repository created after the submission period opened.
- [x] Prior work and last pre-period commit disclosed.
- [x] Relevant open-source license added.
- [x] README includes setup, testing, sample-data, judge-path, Codex, and GPT-5.6 sections.
- [x] Compliance check runs without dependencies.
- [ ] Product installs and runs from a fresh clone.
- [ ] Product tests and deterministic demo smoke test pass.
- [ ] Supported platforms and exact judge testing instructions are final.
- [ ] Repository URL is added to Devpost.

## Product evidence

- [ ] Working, non-trivial vertical slice is complete.
- [ ] GPT-5.6 is used for a material part of the implementation or product behavior.
- [ ] Codex collaboration is documented throughout the build.
- [ ] Key human product, engineering, and design decisions are documented.
- [ ] Every claimed behavior is tied to the submission commit.
- [ ] Real `/feedback` Session ID from the primary build task is recorded.

## Devpost project

- [ ] Project title, entrant-written tagline, and entrant-edited description are final.
- [ ] Submitter type is selected.
- [ ] Eligible country of residence is selected by the entrant.
- [ ] Category is confirmed as exactly one of the allowed choices.
- [ ] Built-with technologies are accurate.
- [ ] Repository link is present.
- [ ] Judge testing URL/instructions are present if applicable.
- [ ] Developer-tool installation, platform, and testing instructions are present.

## Demo video

- [ ] Video is under three minutes.
- [ ] Video shows the project working.
- [ ] Audio explains what was built.
- [ ] Audio explains how Codex was used.
- [ ] Audio explains how GPT-5.6 was used.
- [ ] Video contains no unlicensed music, trademarks, or third-party material.
- [ ] YouTube link works in an incognito window.

## Final submission

- [ ] Latest Devpost announcements and requirements have been rechecked.
- [ ] `npm run submit:check` passes.
- [ ] Team members, if any, have joined and accepted before the deadline.
- [ ] Free judge access will remain available through at least August 12, 2026.
- [ ] OpenAI Build Week submission status is **Submitted**; a published standalone Devpost project page is not sufficient.
